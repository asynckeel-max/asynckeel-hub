# 📈 AsyncKeel KPI Dashboard (Month 3 → Launch)

This dashboard tracks the core marketing and revenue signals from **May 2026** to **July 2026 (launch month)**.

## North Star Metric

**Active users** = GitHub clones (engaged) + paid license customers (Indie + Studio)

---

## Acquisition KPIs

- GitHub stars
- Clone rate (weekly clones / weekly unique visitors)
- Website unique visitors
- Organic search traffic

## Activation KPIs

- README → repo click-through rate
- First commit rate (within 7 days)
- Quick start completion rate

## Revenue KPIs

- Indie license sales (one-time)
- Studio license sales (one-time)
- Early-bird conversion rate

## Retention KPIs

- GitHub watchers
- Newsletter subscriber retention
- Repeat purchase rate (Indie → Studio upgrade)

## Content KPIs

- Blog post views
- Tutorial completion rate
- Social media engagement rate
